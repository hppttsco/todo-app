# To-Do List Application

## Description
This is a simple to-do list application that allows users to manage their tasks effectively. The application utilizes local storage functionality, enabling users to save their tasks even after refreshing the page.

## Features
- Add new tasks with a title and description.
- Mark tasks as completed.
- Delete tasks.
- Persist tasks using local storage, ensuring they remain available after a page reload.

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/hppttsco/todo-app.git
   ```
2. Navigate to the project directory:
   ```bash
   cd todo-app
   ```
3. Open `index.html` in your web browser to run the application.
